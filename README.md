Python-With-Rohan
=================

> Teaching Python to Rohan

### Syllabus

- Getting started
    - Setting up the dev env
    - The command line
    - A text editor
    - Python REPL
  
- Python fundamentals (in iPython notebook)
    - Variables and expressions
    - Types - strings, int, float, etc.
    - Loops
    - Conditionals
    - Functions
    - Lists and Dict

- Understanding the web
    - HTTP
    - Request / Response and Client / Server model 
    - Browsers
    - DNS, IP Adresses, basic networking intro
    - APIs

- Version Control
    - Importance of version control
    - Setting up github account
    - Getting and sharing code for this class

- Making websites - Primer to HTML / CSS
    - HTML markup basics
    - Purely static websites 
    - Chrome console
    - CSS classes / colors / fonts
    - Hosting static websites

- JS 101 - making your apps interactive
    - The document object model
    - Click handlers / alerts / chrome console
    - jQuery goodness - animations

- Building web apps with Flask and Django 
    - routing
    - forms
    - templates and views
    - models and databases

- Looking for help 
    - IRC
    - StackOverflow
    - HackerNews
